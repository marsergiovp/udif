# UDIF Converters

Converters transform platform-specific data exports into valid UDIF files.

## Why Converters Matter

Every major AI platform allows you to export your data — but each 
export is in a proprietary format that only works within that platform's 
ecosystem. Converters solve this by translating those exports into UDIF, 
giving you a file you actually own that any system can read.
